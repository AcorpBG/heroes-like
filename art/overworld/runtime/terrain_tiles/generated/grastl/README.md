# Generated grastl terrain replacement trials

Generated/original replacement trial assets for the HoMM3-style `grastl` terrain atlas.

- `frames_64/`: game-usable 64x64 frame cuts named `00_00.png` through `00_78.png`.
- `source_sheets/`: accepted/generated 1024x1024 2x2 source sheets used to cut the 64x64 frames.
- `previews/`: review contact sheets.
- `experiments/`: earlier prompt experiments and verification images kept for provenance, not runtime use.

Frame mapping currently follows the verified/generated `grastl` catalog structure:
- `00-19`: grass/dirt transitions
- `20-39`: grass/sand transitions
- `40-48`: mixed grass/dirt/sand junctions
- `49-72`: grass interior variants
- `73-78`: embedded dirt/sand and extra mixed bridge stamps
